# 板块指数一句话描述功能

## 功能概述

在主界面"股票指数K线图"板块的各监测板块标题部分添加了"一句话"简要描述，描述信息存储在 `zs_fund_online_ui.json` 配置文件中。

## 数据存储结构

配置文件 `data/zs_fund_online_ui.json` 新增 `zs_descriptions` 字段：

```json
{
  "zs_all": {
    "1.000698": ["科创100", "019861,019867"],
    ...
  },
  "zs_descriptions": {
    "1.000698": "科创100指数是科创板中小市值成长股的代表指数，聚焦半导体、生物医药、新能源等硬科技领域...",
    ...
  },
  "type_all": ["D", "W", "M"],
  "formula_all": ["MACD"],
  "unitWidth": -5
}
```

## 实现文件

1. **utils/IndexDescription.py** - 描述管理工具模块
   - `initialize_descriptions()` - 初始化所有板块描述
   - `get_predefined_description()` - 获取预定义描述
   - `generate_default_description()` - 生成默认描述
   - `load/save_descriptions_from_config()` - 配置文件读写

2. **web_server.py** - Web服务器
   - 启动时自动初始化描述数据
   - 新增API端点：
     - `GET /api/index/descriptions` - 获取所有描述
     - `GET /api/index/description/<code>` - 获取单个描述
     - `PUT /api/index/description/<code>` - 更新单个描述
     - `POST /api/index/descriptions/refresh` - 刷新所有描述

3. **templates/monitor.html** - 监控页面模板
   - 在板块标题下方显示描述
   - 添加描述样式：`.index-description`

## 预定义描述数据

已内置 63 个常用板块指数的详细描述，包括：
- 宽基指数：上证指数、沪深300、中证500、科创50/100等
- 行业指数：半导体、新能源、医药、消费等
- 海外指数：纳斯达克、日经225、恒生科技等

## 首次运行

服务器启动时会自动检查并初始化描述数据：
1. 检查配置文件中是否已有描述
2. 若有缺失，使用预定义数据填充
3. 若无预定义数据，生成默认描述

## 示例

**科创100**（用户示例）：
> 科创100指数是科创板中小市值成长股的代表指数，聚焦半导体、生物医药、新能源等硬科技领域，具有高弹性、高波动特征，是布局科技创新前沿的核心宽基工具。

**上证指数**：
> 上证指数是上海证券交易所编制的综合股价指数，反映沪市整体走势，是中国A股市场最重要的基准指数之一。
