### 5. 【中等】数据库表缺少更新时间触发器 ❌

**位置**：FundDatabase.py 第64-71行

```sql
CREATE TABLE IF NOT EXISTS funds (
    fund_code TEXT PRIMARY KEY,
    fund_name TEXT,
    user_position REAL DEFAULT 0,
    added_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP  -- 没有自动更新
)
```

**问题**：updated_time 字段不会自动更新

**修复建议**：
```sql
-- 添加触发器
CREATE TRIGGER IF NOT EXISTS update_fund_timestamp 
    AFTER UPDATE ON funds
BEGIN
    UPDATE funds SET updated_time = CURRENT_TIMESTAMP WHERE fund_code = NEW.fund_code;
END;
```
