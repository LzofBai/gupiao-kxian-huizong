### 保存估值到CSV

```python
import csv
from FundValuationAPI import FundValuationAPI
from datetime import datetime

api = FundValuationAPI()
funds = ['001593', '001549', '004742']
results = api.batch_calculate_valuations(funds)

# 保存到CSV
filename = f"fund_valuation_{datetime.now().strftime('%Y%m%d')}.csv"
with open(filename, 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['基金代码', '基金名称', '估算净值', '估算涨跌幅', '估算时间'])
    
    for code, val in results.items():
        writer.writerow([
            code,
            val['基金名称'],
            val['估算净值'],
            f"{val['估算涨跌幅']}%",
            val['估算时间']
        ])

print(f"已保存到: {filename}")
```

## 📱 实用小工具

### 工具1: 基金涨跌提醒

创建文件 `fund_alert.py`:

```python
from FundValuationAPI import FundValuationAPI

api = FundValuationAPI()

# 你的基金列表
my_funds = ['001593', '001549', '012842']

results = api.batch_calculate_valuations(my_funds)

print("\n" + "="*50)
print("🔔 基金涨跌提醒")
print("="*50 + "\n")

for code, val in results.items():
    change = val['估算涨跌幅']
    
    # 涨跌超过0.5%则提醒
    if abs(change) > 0.5:
        emoji = "📈" if change > 0 else "📉"
        print(f"{emoji} {val['基金名称']}: {change:+.2f}%")
```

运行:
```bash
python fund_alert.py
```

### 工具3: 定时自动更新

创建文件 `auto_update.py`:

```python
import schedule
import time
from datetime import datetime
from FundValuationAPI import FundValuationAPI

def update_valuations():
    """更新基金估值"""
    print(f"\n[{datetime.now().strftime('%H:%M:%S')}] 更新基金估值...")
    
    api = FundValuationAPI()
    funds = ['001593', '001549', '004742']
    results = api.batch_calculate_valuations(funds)
    
    for code, val in results.items():
        print(f"  {val['基金名称']}: "
              f"{val['估算净值']:.4f} ({val['估算涨跌幅']:+.2f}%)")

# 交易日 9:30-15:00，每30分钟更新一次
schedule.every(30).minutes.do(update_valuations)

print("🤖 自动更新服务已启动")
print("⏰ 每30分钟更新一次")
print("\n按 Ctrl+C 停止")

# 首次立即执行
update_valuations()

# 保持运行
while True:
    schedule.run_pending()
    time.sleep(60)
```

运行:
```bash
python auto_update.py
```

## ❓ 常见问题

**Q: 为什么有的基金获取不到估值?**  
A: 可能是基金代码错误，或该基金暂未公布持仓信息。

**Q: 估值和实际涨跌差距大吗?**  
A: 通常在±0.3%以内，仅供参考。

**Q: 可以增加持仓股票数量吗?**  
A: 可以，修改 `get_fund_top_holdings` 方法中的 `topline` 参数。

**Q: 多久更新一次合适?**  
A: 建议5-30分钟更新一次，避免过于频繁。

**Q: 如何添加更多指标?**  
A: 可以在重仓股明细中加入成交量、市盈率等指标。
