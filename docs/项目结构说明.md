## 📄 核心文件说明

### 1. **web_server.py** - Web服务器主入口
Flask Web应用的主文件，包含所有API路由定义。

**主要路由：**
- `/` - 首页（重定向到监控页面）
- `/api/fund/*` - 基金管理相关API
- `/api/valuation/*` - 估值计算相关API
- `/api/kline/*` - K线图相关API
- `/api/monitor/view` - 监控页面渲染
- `/api/admin` - 管理页面渲染

**关键配置：**
```python
CONFIG_FILE = 'data/zs_fund_online.json'  # 配置文件路径
```

### 2. **api/FundValuationAPI.py** - 基金估值核心
基金估值功能的核心实现，包含数据获取和计算逻辑。

**主要功能：**
- 从天天基金网获取基金信息
- 从东方财富网获取持仓数据
- 实时查询股票行情
- 并发优化（ThreadPoolExecutor 5线程）
- 数据缓存机制

**数据来源：**
- 基金信息：`http://fundgz.1234567.com.cn/js/{fund_code}.js`
- 持仓数据：`http://fundf10.eastmoney.com/FundArchivesDatas.aspx`
- 股票行情：`http://push2.eastmoney.com/api/qt/ulist.np/get`

### 3. **api/KLineAPI.py** - K线图API
K线图URL生成和下载功能封装。

**URL格式：**
```
http://webquoteklinepic.eastmoney.com/GetPic.aspx?
nid={市场代码}.{股票代码}&
imageType=KXL&
token=44c9d251add88e27b65ed86506f6e5da&
kline={周期}&
...
```

### 4. **utils/Logger.py** - 日志工具
自定义日志记录器，支持日志滚动和多级别日志。

**特性：**
- 日志轮转（默认10MB，保留5个备份）
- 多级别日志（DEBUG, INFO, WARNING, ERROR, CRITICAL）
- 自动创建日志目录

### 5. **scripts/txt2str.py** - 文件转换工具
通用文件读取和JSON解析工具。

**主要函数：**
- `getcontext()` - 自动识别编码并读取文件
- `file2json()` - 文件转JSON对象

## 🔄 数据流程

### 基金估值计算流程：
```
用户请求
    ↓
web_server.py (/api/valuation/batch)
    ↓
FundValuationAPI.calculate_fund_valuation()
    ↓
├─ get_fund_top_holdings() → 获取持仓（优先本地缓存）
├─ get_stock_realtime_quote() → 并发获取股票行情（5线程）
└─ 加权计算估值
    ↓
返回结果 {估值, 涨跌幅, 单日盈亏, ...}
```

### 基金添加流程（预览-确认机制）：
```
用户输入基金代码
    ↓
前端调用 /api/fund/preview/{code}
    ↓
├─ 验证基金代码格式（6位数字）
├─ 联网验证基金是否存在
├─ 获取前十重仓股
└─ 返回预览数据
    ↓
用户确认 → 前端调用 /api/fund/add
    ↓
添加到监控列表 + 初始化持仓金额
```

## 🔍 开发建议

### 添加新API模块
1. 在`api/`目录创建新的Python文件
2. 在`api/__init__.py`中导出新模块
3. 在`web_server.py`中导入并使用

### 添加新工具函数
1. 在`utils/`目录创建新的Python文件
2. 在`utils/__init__.py`中导出新函数
3. 在需要的地方导入使用

### 修改配置路径
所有路径相关配置都在各文件的开头部分，统一使用相对于项目根目录的路径。
